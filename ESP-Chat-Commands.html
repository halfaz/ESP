Adds Extra Chat Commands

Code Preview

on skript load: set {_filename} to "ESP-Chat-Commands.sk" if {ESP::loaded_scripts::*} doesn't contain {_filename}: add {_filename} to {ESP::loaded_scripts::*} command /clearchat []: aliases: /cc permission: chat.clear permission message: %{ESP::name}% %{ESP::e1}%You don't have permission to clear chat! trigger: delete {chat::*} loop 100 times: broadcast " " broadcast "%{ESP::name}% %{ESP::t1}%Chat has been cleared by %{ESP::h1}%%player%" command /online: aliases: /who trigger: set {_count} to 0 send "%{ESP::name}%" send "&8&l&m--------------------------------" send "%{ESP::h1}%Online Players:" loop all players: add 1 to {_count} send "%{ESP::t1}%- %{ESP::h1}%%loop-player%" send "%{ESP::t1}%Total online: %{ESP::h1}%%{_count}%" send "&8&l&m--------------------------------"
