ESP-Welcome

About This Skript

Handles player welcome messages and join/leave notifications.

Code Preview

on skript load: set {_filename} to "ESP-Welcome.sk" if {ESP::loaded_scripts::*} doesn't contain {_filename}: add {_filename} to {ESP::loaded_scripts::*} on first join: wait 1 second send "____________________________________________________" to player send "%{ESP::t1}%Welcome %{ESP::h1}%%player% %{ESP::t1}%to %{ESP::name}%!" to players on join: if player has played before: wait 1 second send "____________________________________________________" to player send "%{ESP::t1}%Welcome %{ESP::h1}%%player% %{ESP::t1}%to %{ESP::name}%!" to players send "%{ESP::h1}%&l%{ESP::w1}%:" to player send "%{ESP::t1}%• %{ESP::w2}%" to player send "%{ESP::t1}%• %{ESP::w3}%" to player send "%{ESP::t1}%• %{ESP::w4}%" to player send "____________________________________________________" to player send "%{ESP::h1}%&lUp and Coming:" to player send "%{ESP::t1}%• %{ESP::w5}%" to player send "%{ESP::t1}%• %{ESP::w6}%" to player send "%{ESP::t1}%• %{ESP::w7}%" to player send "____________________________________________________" to player
