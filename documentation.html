Getting Started

Installation Guide

  1. Install Skript Plugin

    Navigate to your server's plugin folder and download Skript

    Restart your server after installation

  2. Install ESP Scripts
    1. Go to the ESP home page and navigate to "Script Lists"
    2. Choose your desired ESP script
    3. Create a new file named ESP-[scriptname].sk
    4. Copy the script content into your new file
    5. Run /sk reload all in your console
  3. Install ESP Settings

    ESP Settings is required for all scripts to function properly:

    1. Find ESP-Settings.sk in the script list
    2. Follow the same installation process as above
    3. Check script descriptions for required dependencies
    4. Add any required variables to ESP-Settings.sk
  4. Configuration

    In ESP-Settings.sk, you can configure:

    • Server name
    • Message colors
    • Script-specific variables
  5. Quick Reload Feature

    For faster reloading:

    1. Install ESP-FastCommands.sk
    2. Use command ESPreload to quickly reload all active scripts
